# Projeto Expo 👋
 
Este é um projeto [Expo](https://expo.dev) simplificado.
 
## 🚀 Como começar
 
Siga estas etapas para rodar o aplicativo:
 
1. **Instalar as dependências:**
   ```bash
   npm install
Iniciar o servidor:
 
Bash
 
npx expo start
No terminal, você verá um QR Code. Escaneie-o com o aplicativo Expo Go no seu celular para ver o app rodando em tempo real.
 
📂 Estrutura de Pastas
Para começar a programar, edite os arquivos dentro da pasta:
 
app/: É aqui que você cria as telas e a lógica do seu app.
 
🧹 Limpar o Projeto
Se você quiser deletar o código de exemplo e começar um projeto do zero, rode o comando abaixo:
 
Bash
 
npm run reset-project
Este comando move o código inicial para a pasta app-example e cria uma pasta app vazia para você.
